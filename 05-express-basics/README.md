# 01-Express Js

This is a simple demonstration of building a web server using Express.js in TypeScript.

## What is Express.js?
Express is a fast, unopinionated, and minimalist web framework for Node.js. It simplifies building web servers and APIs by providing robust routing and HTTP utility methods.

## Features in this example:
- **Routing**: Setting up routes for different URL paths (`/`, `/contact-us`, `/tweets`).
- **HTTP Methods**: Handling different HTTP methods like `GET` (for fetching data) and `POST` (for sending/creating data).
- **Responses**: Sending back text responses and HTTP status codes (`200 OK`, `201 Created`).

## Running the code:
```bash
npm start
# Or run with tsx:
npx tsx index.ts
```
